- 🎨 Modern UI - Clean and intuitive interface

## Getting Started

### Option 1: Open Directly in Browser

Simply open `index.html` in your web browser:

```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

### Option 2: Using a Local Web Server

For a better development experience, use a local web server:

**Using Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Using Node.js (http-server):**
```bash
npx http-server -p 8000
```

**Using PHP:**
```bash
php -S localhost:8000
```

Then open your browser and navigate to `http://localhost:8000`

## Customization

You can easily customize the web interface by modifying:

- **Colors**: Edit CSS variables in `style.css` (`:root` section)
- **Content**: Modify text and sections in `index.html`
- **Functionality**: Add or modify JavaScript in `script.js`
